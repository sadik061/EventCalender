Welcome to the user guide

A user guide for DGNM's "Event Calendar Application"

Calendar

This is the Dashboard of this Application. Here we have included every steps of creating events, customizing, shifting and multiple views of event on calendar views.

1. Calendar view: Without any events.

2. Create Events: Create events from Dashboard.

3. Insert Events Information: Insert event Information including - event name, date, venu, funded by, etc. For next attempt, Please click "Choose Participant" button to add Participant (Instructors/Resource person).

1. Search Institute: Search any "institutes" from database. You can even search by area.

2. Add Participants: Add any "Participant" from choosen Institute. You can choose participants as "Role" as given under the Assign button.

3. Participants Tabs: Here are two participant tabs, first one is "Participant Tab" and second one is "Resource Person Tab". You can add new Resource Person from "Resource Person Tab".
